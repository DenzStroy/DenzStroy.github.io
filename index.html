Pelajaran Front-End Development

Belajar cara membangun antarmuka pengguna yang menarik dan responsif.

Pengantar HTML

HTML (HyperText Markup Language) adalah bahasa dasar untuk membangun struktur halaman web.

Styling dengan CSS

CSS (Cascading Style Sheets) digunakan untuk memberikan tata letak dan desain pada halaman web.

JavaScript Dasar

JavaScript memberikan interaktivitas dan fungsionalitas dinamis ke dalam situs web.

Framework Front-End

Belajar menggunakan framework seperti React, Angular, atau Vue untuk membangun aplikasi web modern.
